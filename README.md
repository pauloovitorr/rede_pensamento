# rede_pensamento
Projeto desenvolvido com o objetivo de treina Node.js e express.
Este projeto foi possível trabalhar com criação de sessões e cookies, autentiação de usuário, bastante CRUD e uma boa lógica de prograçamação.

Para rodar o projeto certifique-se de ter o node.js instalado na sua máquina, XAMPP para criar o ambiente Mysql e de preferência o Mysql Workbench para acompanhar os banco de dados e suas tabelas. Após isso crie um banco de dados com o nome "projeto_pensamentos", usuário "root" e senha vazia. Posteriormente execute o comando "npm install" para instalar todas as dependências necessárias e executar o comando "npm start" para iniciar a aplicação.

Projetinho desenvolvido no curso "Node.js do Zero a Maestria" do Matheus Battisti e com diversas modificações minhas.
